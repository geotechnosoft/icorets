Payslip Portal Odoo App helps users view and manage payslips from portal websites. Portal users can search payslips by date from to and batch on portal view, They can apply different sorting by an option like newest, ascending order: month, and descending order: month. Portal users can download payslip reports in PDF format and also send messages and attachments using payslip chatter on portal view.

Features

Manage Payslips on Portal View

User can easily view and manage payslips on portal view.

Payslips Sort By Option

User can sort payslips by Newest, Ascending Order:Month, and Descending Order:Month.

Search Payslip on Portal View

User can search payslip by date from-to and batch on portal view.

Payslip Details on Form View

User can see payslip details in form view.

Download Payslip Report from Portal View
